这里记录一些自己常用的markdown语法

<!-- more -->

## 1.标题
示例
```
# 这是一级标题
## 这是二级标题
...
###### 这是六级标题
```
效果如下:
# 这是一级标题
...
###### 这是六级标题

## 2.代码
示例
```
(```) #去掉左右括号
System.out.println('haha')
(```) #去掉左右括号
```
效果如下
```
System.out.println('haha')
```

## 3.引入
示例
```
>这是引入的示例
```
效果如下
>这是引入的示例
